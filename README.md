# Unictris - Rusty Glyph Edition

Tetris for the terminal - Unicode TUI based game implemented with rust/crossterm.

```
% cargo run
```

Controls: 
* q to quit
* Arrow Left & Arrow Right to move sideways
* Arrow Up to rotate
* Arrow Down to drop
* Space to pause
